El desarrollo de aplicaciones de Internet de las Cosas requiere una serie de tecnologías y soluciones.

El software libre juega un papel muy importante, proporcionando tanto una arquitectura hardware y software de código abierto, como entornos de desarrollo que permiten la creación de aplicaciones para Internet de las Cosas

Mango es una aplicación web, M2M desarrollada en Ajax, que permite a los usuarios acceder y controlar sensores electrónicos, dispositivos y máquinas a través de múltiples protocolos simultáneamente.

Searched Query Mango ofrece una interfaz con la que se pueden crear y configurar que diversas fuentes de datos al mismo tiempo.
También permite la gestión del acceso de los usuarios, aletas, registro de datos y automatización.

Mango permite visualizar, graficar, generar alertas y reportes sobre los datos de sensores, equipos, bases de datos, páginas web, etc.

Puede recibir datos a través de múltiples protocolos en intervalos inferiores a un segundo. Protocolos como BACnet, Modbus, SNMP, etc. Mango es multiplataforma y se puede instalar en Windows, Linux o Mac.
